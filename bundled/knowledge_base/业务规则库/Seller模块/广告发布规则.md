# Seller模块 - 广告发布业务规则

## 1. 功能概述
- **功能描述**: 卖家通过Post Ad功能创建并发布For Sale类目广告,支持AI辅助填充和手动填写两种方式
- **用户角色**: 卖家(Seller)
- **入口位置**: 
  - 首页"Post an ad"按钮
  - 导航栏"Post Ad"入口
  - 管理页"Post another ad"按钮
- **依赖模块**: 
  - 用户认证模块(登录Session)
  - 照片上传服务(CDN存储)
  - AI生成服务(标题/描述生成)
  - 支付模块(推广套餐支付)

## 2. 核心流程
### 2.1 主流程
**标准发布流程(含AI辅助)**:
1. 用户登录并进入Post Ad页面 `/postad`
2. 选择类目(浏览Browse或搜索Search)
3. 上传照片(0-20张,触发AI识别)
4. AI自动生成标题和描述(骨架屏加载30秒内)
5. 用户确认或修改AI生成内容
6. 填写必填字段(Price, Location, Condition等)
7. 选择可选字段(Item Specifics, Contact, 推广套餐)
8. 点击"Post my Ad"提交
9. 前端验证通过后调用后端API
10. 跳转成功页面(若选择推广则跳转支付页)

**手动发布流程(无AI辅助)**:
1-2. 同上
3. 用户手动输入标题(不上传照片或AI生成失败)
4. 手动填写描述(≥15字符)
5-10. 同上

### 2.2 异常流程
- **AI生成超时**: 30秒未返回 → 显示错误提示"AI generation failed. Please enter manually." → 允许手动填写
- **照片上传失败**: 网络错误/文件过大 → 重试3次 → 失败后提示用户手动删除重试
- **必填字段验证失败**: 提交时校验 → 页面滚动到第一个错误字段 → 显示红色错误提示 → 禁止提交
- **Session过期**: 填写过程中Session失效 → 保存草稿 → 跳转登录页 → 登录后恢复草稿
- **提交API失败**: 后端返回500 → Toast提示"提交失败,请稍后重试" → 保留用户数据 → 可重试

## 3. 业务规则

### 3.1 输入规则

| 字段 | 类型 | 必填 | 长度/限制 | 格式 | 默认值 | 说明 |
|-----|------|-----|---------|------|--------|------|
| Photos | File[] | 否(引导) | 0-20张 | jpg/png/gif/webp,单张≤10MB | - | 首张触发AI识别,可拖拽调整顺序 |
| Ad Title | String | 是* | 无明确限制(推测≤100字符) | 任意文本 | AI生成 | 支持AI生成,用户可修改 |
| Description | String | 是* | 15-10000字符 | 任意文本 | AI生成 | 实时字数统计,支持换行 |
| Category | String | 是(隐式) | 4级类目路径 | 面包屑格式 | - | 必须选择叶子节点,显示完整路径 |
| Price | Decimal | 是* | ≥0.01, ≤999999.99 | 数字+最多2位小数 | - | 货币符号固定为£(英镑) |
| Location | Object | 是* | 包含city+postcode | {city, area, postcode} | 用户账户位置 | 隐私保护:仅显示city+area |
| Condition | Enum | 是(Required) | - | New/Used/For parts or not working | - | Item Specifics必填字段 |
| Contact Email | String | 是(默认勾选) | - | Email格式 | 用户注册邮箱 | 不对外显示,仅平台站内信通知 |
| Contact Phone | String | 否 | 11位 | UK手机号(07开头) | - | 可选添加,公开显示给买家 |
| Show Map | Boolean | 否 | - | true/false | false | 勾选后显示Google Map近似区域 |
| Featured | Object | 否 | {selected, days} | days: 3/7/14 | {false, 7} | 推广套餐:置顶广告,可选天数 |
| Urgent | Boolean | 否 | - | true/false | false | 推广套餐:显示"Urgent"标签,固定7天 |
| Spotlight | Boolean | 否 | - | true/false | false | 推广套餐:首页展示,固定7天(可能需解锁) |
| Youtube Link | Boolean | 否 | - | YouTube URL | false | 可选付费功能(£1.99),需验证URL |

### 3.2 校验规则

**前端实时验证(blur时触发)**:
- **Description字符数**: 
  - `<15字符` → 显示"15 characters minimum"(红色)
  - `15-10000字符` → 通过
  - `>10000字符` → 自动截断或禁止输入
  - 计数规则:按character计算,emoji算1个字符

- **Price格式**:
  - 仅允许输入数字和小数点
  - 最多2位小数
  - 不允许负数(输入时自动过滤)
  - `<0.01` → 错误提示"最低价格为£0.01"

- **Phone格式**:
  - UK手机号:07开头,共11位数字
  - 格式示例:"07123456789"
  - 非UK号码 → 错误提示"请输入有效的UK手机号"

- **YouTube链接验证**:
  - 域名必须为`youtube.com`或`youtu.be`
  - 其他URL → 错误提示"请输入有效的YouTube链接"

**提交时全局验证**:
- 必填字段不能为空:Photos(引导但非强制)/Title/Description/Category/Price/Location/Condition/Contact(至少勾选一个)
- 若有字段错误:
  1. 页面滚动到第一个错误字段
  2. 字段边框变红
  3. 显示错误提示文本(supporting-text)
  4. Toast提示:"Please complete all required fields"
  5. 禁止提交,按钮保持可点击状态

### 3.3 权限规则
- **登录用户权限**:
  - ✅ 可发布广告
  - ✅ 可选择推广套餐
  - ✅ 可保存草稿
  - ✅ 可上传照片

- **游客权限**:
  - ❌ 无法进入Post Ad页面
  - 跳转登录页:URL重定向`/login?redirect=/postad`

- **Session管理**:
  - Session有效期:24小时(推测)
  - 过期处理:保存草稿 → 跳转登录 → 登录后恢复
  - Session复用:首次登录保存Session到本地 → 后续测试直接加载 → 节省15秒登录时间

### 3.4 业务约束

- **照片约束**:
  - 数量限制:0-20张
  - 尺寸限制:单张≤10MB(推测),总计≤100MB(推测)
  - 格式限制:jpg/png/gif/webp
  - 顺序调整:长按拖拽调整,第一张为主图(缩略图)
  - 删除操作:点击X直接删除,无二次确认

- **类目约束**:
  - 必须选择到叶子节点(最深层级)
  - 不能选择中间层级(如"For Sale"根节点)
  - 示例:✅ "For Sale > Baby & Kids Stuff > Car Seats" / ❌ "Baby & Kids Stuff"
  - AI推荐标签:最多显示5个(推测),用户可点击快速选择

- **AI生成约束**:
  - 触发条件:上传首张照片 OR 输入标题后选择类目
  - 生成时长:≤30秒,超时视为失败
  - 内容可修改:用户可随时编辑AI生成的标题/描述
  - 重新生成:点击"Change"按钮可重新调用AI生成

- **推广套餐约束**:
  - 可多选:用户可同时选择Featured+Urgent+Spotlight
  - 价格累加:总费用=各套餐价格之和
  - Featured天数:3天(£0.50)/7天(£1.00)/14天(£1.80),默认7天
  - Urgent价格:固定£0.75,固定7天
  - Spotlight价格:固定£1.99,固定7天(可能需满足解锁条件)

- **草稿约束**:
  - 自动保存时机:每30秒(推测) + 用户退出时
  - 草稿有效期:30天(推测)
  - 草稿存储:云端绑定账户(支持跨设备恢复)
  - 恢复优先级:最近一次保存的草稿

- **站点差异**:
  - 货币符号:固定£(英镑,UK站)
  - 手机号格式:UK手机号(07开头)
  - 地图服务:Google Maps(UK地区)

### 3.5 AI辅助生成规则(Genesis Phase 1 - iOS)

**触发场景**:
- **场景A:Photo First**:
  1. 用户上传第一张照片
  2. AI识别物品类型
  3. 自动生成:标题 + 描述(15-10000字符) + 推荐类目标签(3-5个)
  
- **场景B:Title First**:
  1. 用户输入标题(如"Vintage red leather jacket")
  2. AI推荐类目列表
  3. 用户选择类目后进入表单
  4. AI补充生成完整描述

**生成过程**:
- **加载状态**(骨架屏):
  - Title字段:1行灰色矩形(宽245px,高20px)
  - Description字段:3行灰色矩形(宽度递减:285px/245px/185px)
  - 骨架屏呈现呼吸动画(opacity 0.3-0.6循环,1.5s)
  - 右上角显示"01"旋转图标
  
- **生成成功**:
  - 骨架屏淡出(fade-out, 0.3s)
  - AI内容填充
  - 用户可立即编辑
  - "Change"按钮可见(可重新生成)

- **生成失败**(超时>30秒 OR 服务不可用):
  - 显示错误提示:"AI generation failed. Please enter manually."
  - 允许用户手动填写
  - "Change"按钮变为"Retry"

**内容质量约束**:
- 标题:简洁描述物品,包含关键属性(如颜色/品牌/型号)
- 描述:≥15字符,符合Gumtree风格,包含物品状态/特性/购买建议
- 无敏感词:AI生成内容需经过内容审核,避免违禁词汇

## 4. 错误处理

| 错误码 | 错误信息 | 触发条件 | 用户提示 | 处理方式 |
|-------|---------|---------|---------|---------|
| ERR_TITLE_EMPTY | Title is required | Title字段为空提交 | "Ad Title" + "supporting-text"(红色) | 滚动到Title字段,高亮边框 |
| ERR_DESC_MIN_LENGTH | Description too short | Description<15字符 | "15 characters minimum" | 实时提示,禁止提交 |
| ERR_DESC_MAX_LENGTH | Description too long | Description>10000字符 | "10000 characters maximum" | 自动截断或禁止输入 |
| ERR_PRICE_INVALID | Invalid price | Price为负数/非数字 | "Please enter a valid price" | 输入时过滤非法字符 |
| ERR_PRICE_MIN | Price too low | Price<£0.01 | "Minimum price is £0.01" | 提交时验证 |
| ERR_CATEGORY_REQUIRED | Category not selected | Category未选择 | "Please choose a category"(红色) | 滚动到Category字段 |
| ERR_LOCATION_REQUIRED | Location is required | Location为空 | "supporting-text" | 滚动到Location字段 |
| ERR_CONDITION_REQUIRED | Condition not selected | Condition未选择 | "supporting-text" | 滚动到Condition字段 |
| ERR_CONTACT_REQUIRED | Contact info required | Email和Phone均未勾选 | "Please select at least one contact method" | 滚动到Contact字段 |
| ERR_PHOTO_UPLOAD_FAILED | Photo upload failed | 网络错误/文件过大 | "Upload failed. Please try again." | 显示重试按钮,自动重试3次 |
| ERR_AI_TIMEOUT | AI generation timeout | AI生成>30秒无响应 | "AI generation failed. Please enter manually." | 允许手动填写,显示"Retry"按钮 |
| ERR_SESSION_EXPIRED | Session expired | 填写过程中Session过期 | "Your session has expired. Please log in again." | 保存草稿,跳转登录页 |
| ERR_SUBMIT_FAILED | Submit failed | 后端API返回500 | "Failed to post ad. Please try again later." | Toast提示,保留数据,可重试 |
