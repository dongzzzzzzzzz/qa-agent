# Pay & Ship - 广告发布业务规则

## 1. 功能概述
- **功能描述**: 卖家在 Gumtree 平台发布支持配送的二手商品广告，通过 Enable delivery 开关开启 Pay & Ship 功能
- **用户角色**: 卖家（Seller）
- **入口位置**: 首页 → 发布广告 → 填写商品信息表单
- **依赖模块**: 商品类目模块、价格模块、物流配送模块、MangoPay 支付模块

## 2. 核心流程

### 2.1 主流程
1. 卖家登录账号，进入广告发布页面
2. 选择开放 Pay & Ship 的商品类目（如 for-sale > clothing）
3. 填写商品标题、价格（1-250英镑）、描述、图片
4. 系统判断类目和价格区间，若满足条件则显示"Enable delivery"开关
5. 卖家勾选"Enable delivery"开关
6. 卖家选择包裹尺寸（小/中/大）
7. 点击"Post ad"发布广告
8. 跳转到广告详情页，显示 Buy now 按钮和配送费用

### 2.2 异常流程
- 类目未开放：Enable delivery 开关不可见，广告不支持 Pay & Ship
- 价格超出区间：Enable delivery 开关不可见（价格 < 1 或 > 250 英镑）
- 勾选 Enable delivery 但未选包裹尺寸：提交被阻止，显示"Please select parcel size"错误提示
- 广告发布失败：页面提示错误信息，留在发布页面

## 3. 业务规则

### 3.1 输入规则
| 字段 | 类型 | 必填 | 限制 | 格式 | 说明 |
|------|------|------|------|------|------|
| 商品标题 | String | 是 | - | 文本 | 商品名称 |
| 商品价格 | Number | 是 | [1, 250] 英镑 | 数字 | Enable delivery 开关展示的价格条件 |
| 商品图片 | File | 是 | 至少1张 | 图片文件 | 商品照片 |
| 商品描述 | String | 否 | - | 文本 | 商品详细描述 |
| Enable delivery | Boolean | 否 | - | 开关 | 是否开启配送功能 |
| 包裹尺寸 | Enum | 条件必填 | 小/中/大 | 枚举 | 勾选 Enable delivery 后必填 |

### 3.2 校验规则
- Enable delivery 可见条件：类目为 Pay & Ship 开放类目（如 for-sale > clothing）**且**商品价格在 [1, 250] 英镑范围内
- 开启 Enable delivery 后，包裹尺寸必须选择，否则阻止发布
- 价格边界值：1英镑包含（≥1），250英镑包含（≤250）

### 3.3 权限规则
- 仅已登录的卖家账号可以发布广告
- 发布者无法对自己发布的广告点击"Buy now"

### 3.4 业务约束
- Pay & Ship 仅支持特定开放类目（非所有类目），当前主要类目：for-sale > clothing
- 商品价格超过 250 英镑不可开启配送，限制避免高价商品的配送风险
- 包裹尺寸影响配送费用，卖家选择后不可在订单创建后更改

### 3.5 My Ads 列表标签展示规则
| 广告状态 | 标签展示 |
|---------|---------|
| 非开放类目 | 不展示任何配送标签 |
| 开放类目 + 未开启配送 | 展示"Eligible for delivery" |
| 开放类目 + 已开启配送 | 展示"Delivery enabled" |
- 页面顶部推荐开启配送的数量：仅统计"Eligible for delivery"标签数量（不包含已开启和非类目广告）

### 3.6 广告详情页配送费用展示
| 配送方式 | 小包裹 | 中包裹 | 大包裹 |
|---------|--------|--------|--------|
| 配送到家 | £2.99 | £3.99 | £6.59 |
| 配送到自提点 | £2.59 | £2.99 | £3.49 |
- 买家广告详情页展示 Buy now 按钮条件：广告支持配送 **且** 访问者非广告发布者本人

## 4. 错误处理
| 错误场景 | 用户提示 | 触发条件 |
|---------|---------|---------|
| 未选包裹尺寸 | Please select parcel size | 勾选 Enable delivery 后未选尺寸直接发布 |
| 价格不在区间 | Enable delivery 不可见 | 价格 < 1 或 > 250 英镑 |
| 非开放类目 | Enable delivery 不可见 | 选择非 Pay & Ship 类目 |
