## 📊 Application Overview

### 功能定位
Gumtree 是英国最大的分类广告网站，主页用于展示最新的商品和服务广告。第三方广告位是网站的主要收入来源之一。

### 广告位业务规则
1. **广告位预留机制**：页面加载时预留广告位容器，即使广告未投放也保持容器存在
2. **异步加载**：第三方广告通过异步脚本加载，不阻塞主内容展示
3. **响应式设计**：桌面端展示顶部横幅和右侧广告，移动端可能调整布局
4. **数据标记**：所有广告位包含 `data-display-ad="true"` 属性用于识别
5. **独立容器**：每个广告位使用独立 div 容器，通过 ID 唯一标识

### 广告位状态枚举
- **已投放已加载**：广告位有尺寸，内容展示正常
- **已预留未投放**：广告位存在但尺寸为 0（width: 0, height: 0）
- **加载失败**：广告脚本加载失败，容器为空
- **被屏蔽**：广告拦截器阻止广告加载

## 📝 测试总结

### 已验证功能（实测 ✅）
1. **广告位容器存在性**：所有 4 个广告位容器均存在于 DOM 中
2. **顶部广告位布局**：位置、尺寸、可见性均正常
3. **数据属性标记**：所有广告位包含 `data-display-ad="true"` 属性
4. **ID 唯一性**：所有广告位 ID 唯一，无冲突
5. **控制台错误**：存在广告相关错误，但不影响主功能
6. **广告内容加载**：iframe、图片等广告元素加载正常（需先接受 Cookie 协议）
7. **广告尺寸合规**：iframe 尺寸符合 IAB 标准（970x250, 728x90 等）
8. **广告交互性**：广告包含可交互元素或 iframe
9. **加载时间性能**：广告加载不阻塞主内容展示
10. **响应式布局**：桌面端、平板端、移动端布局适配正常

### 待验证功能（推断 ⚠️）
1. **右侧广告位投放状态**：当前未投放，需在广告投放时段测试
2. **移动端响应式布局**：需要调整视口或使用真实设备测试
3. **广告拦截器场景**：需要安装插件并测试
4. **页面加载性能**：需要使用 Lighthouse 等工具测量
5. **广告脚本异步加载**：需要分析网络请求瀑布图

### 风险识别
1. **广告投放不稳定**：广告可能因地域、时间、预算等原因不投放
2. **第三方依赖**：广告系统依赖外部 SDK，可能因网络或服务故障失败
3. **浏览器兼容性**：不同浏览器的广告展示可能有差异
4. **隐私法规影响**：Cookie 政策变化可能影响广告投放

### 自动化建议
1. **可自动化场景**：
   - 广告位容器存在性检查（TC-3PA-001, 006, 010）
   - 广告位尺寸和位置验证（TC-3PA-002, 003）
   - 数据属性验证（TC-3PA-013, 014）
   - 响应式布局验证（TC-3PA-017, 018）
   - **广告内容加载验证（TC-3PA-021~026）** 🆕
   - iframe 加载状态检查（TC-3PA-022）
   - 广告尺寸合规性（TC-3PA-023）
   - 广告交互性验证（TC-3PA-024）
   - 广告加载时间性能（TC-3PA-026）

2. **难以自动化场景**：
   - 广告内容审核（需人工判断合规性）
   - 广告点击行为（可能触发计费）
   - 广告投放策略（依赖外部系统）

3. **推荐测试策略**：
   - 使用 mock 广告服务器或测试广告账号
   - 定期检查广告位容器完整性
   - 监控控制台错误，但不作为失败条件
   - 在多种设备和网络环境下测试
