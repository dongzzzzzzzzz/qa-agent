# Bark integration（Web SRP）测试用例

> **生成时间**: 2026-03-17  
> **范围**: Gumtree Services SRP（Web），接入 Bark 三方广告数据源后以广告形式展示  
> **默认测试环境**: `https://www.unicorn.gumtree.io/business-services`（可将 `unicorn` 替换为 `bixi`、`zoidberg` 等）  
> **实验 Header**: `gt_gb_exp_ovr`（线上仅保留 `e_002:B` 新实验；老实验 `e_002:A` 已下线，相关用例不再覆盖）  
> **广告规则**: Gumtree 广告优先；Gumtree 广告在当前 query 下耗尽后展示 Bark 广告  
> **Bark 识别口径（UI）**: 卡片按钮包含 **"Request a call"** / **"Request a call back"** 即 Bark；否则为 Gumtree  
> **卡片展示字段**: 仅展示 **Location** + **Category**  
> **落地页**: Web 端点击卡片进入 H5（埋点不在本批范围）
