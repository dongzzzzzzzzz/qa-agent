# 用例格式参考

## 优先级

- 高：核心功能、高频路径
- 中：次要功能
- 低：边缘场景

## 测试类型

功能 / 边界 / 异常 / 兼容 / 性能 / 安全

## 表格示例

| 编号 | 模块 | 用例标题 | 前置条件 | 测试步骤 | 预期结果 | 优先级 | 测试类型 |
|------|------|----------|----------|----------|----------|--------|----------|

## XMind 层级示例

```markdown
# 登录模块
## 手机号登录
### 验证正确手机号登录成功
- 前置条件：已注册账号
- 测试步骤：输入合法手机号与密码，点击登录
- 预期结果：进入首页
- 优先级：高
- 测试类型：功能
```
